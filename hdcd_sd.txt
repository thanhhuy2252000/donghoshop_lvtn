*Hướng Dẫn Cài Đặt và Sử Dụng Ứng Dụng Web Bán Đồng Hồ Với XAMPP

*Yêu Cầu Hệ Thống
- XAMPP (bao gồm Apache, PHP, và MySQL)
- Composer
- Git
-----------------------
*Cài Đặt
---------------
Bước 1: Cài Đặt môi trường XAMPP
Nếu bạn chưa cài đặt XAMPP, tải và cài đặt XAMPP từ trang chính thức XAMPP:
 https://www.apachefriends.org/download.html
(sau khi cài đặt xamp hãy đảm bảo bạn cài đặt, chạy Apache và MySQL.)
Bước 1.1: Cài Đặt Git
Tải và cài đặt Git từ trang chính thức Git: https://www.git-scm.com/downloads
Bước 1.2: Cài Đặt Composer
Tải và cài đặt Composer từ trang chính thức: https://getcomposer.org/download/

Bước 2: download source: Clone Repository bằng git
Mở git bash ngoài destop máy, hoặc Terminal hoặc Command Prompt trên Windows 
và sao chép mã nguồn từ GitHub về thư mục htdocs trong thư mục cài đặt XAMPP của bạn.

cd C:\xampp\htdocs
git clone https://github.com/thanhhuy2252000/donghoshop_lvtn.git
cd donghoshop_lvtn

Bước 3: Cài Đặt Các Phụ Thuộc: Sử dụng Composer để cài đặt tất cả các phụ thuộc cần thiết cho ứng dụng.
Mở Terminal hoặc Command Prompt trên Windows
"composer install"
-cài laravel bằng composer
"composer global require laravel/installer"

Bước 4: Tạo Cơ Sở Dữ Liệu
Mở phpMyAdmin từ XAMPP (truy cập http://localhost/phpmyadmin), tạo cơ sở dữ liệu mới với tên donghoshop_db.
load file "donghoshop_db.sql" và database	

Bước 5: Cấu Hình Môi Trường của laravel
Sao chép tệp cấu hình môi trường mẫu và chỉnh sửa các thông số cần thiết.
 mở .env thay đổi cấu hình:  

DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=donghoshop_db
DB_USERNAME=root
DB_PASSWORD=
